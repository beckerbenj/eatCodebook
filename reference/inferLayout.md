# Infer layout column in a variable information.

Temporary function: Infers the layout numeric code from the `SPSS`
format of a variable in the data and the information in the
`inputForDescriptives`.

## Usage

``` r
inferLayout(varInfo, GADSdat, inputForDescriptives)
```

## Arguments

- varInfo:

  `varInfo` object.

- GADSdat:

  `GADSdat` object.

- inputForDescriptives:

  Input for descriptive statistics calculation.

## Value

Returns the modified variable information.
