# Create missing value information.

Create information on value level (value labels and missing codes).

## Usage

``` r
createMissings(GADSdat, inputForDescriptives)
```

## Arguments

- GADSdat:

  A `GADSdat` object.

- inputForDescriptives:

  Input for descriptive statistics calculation.

## Value

Returns the value information.
