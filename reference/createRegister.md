# Create register template.

Create register template based on variable information.

## Usage

``` r
createRegister(inputForDescriptives, keywordList = NULL)
```

## Arguments

- inputForDescriptives:

  Object (either `list` or `data.frame`) containing variable
  information.

- keywordList:

  Character vector of keyword columns to be added. If `NULL`, no
  additional columns are added. If `inputForDescriptives` is a `list`,
  `keywordList` must also be a `list` (of character vectors).

## Value

Register template.
