# Embed a Latex Snippet.

Embed a Latex Snippet in the Latex environment used by
[`codebook`](https://beckerbenj.github.io/eatCodebook/reference/codebook.md)..

## Usage

``` r
embedLatexSnippet(snippet)
```

## Arguments

- snippet:

  A Latex snippet.

## Value

A testable latex script.

## Examples

``` r
latexSnippet <- embedLatexSnippet("This is a sentence.")
```
