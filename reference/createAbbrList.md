# Create abbreviation list template.

Create abbreviation list template.

## Usage

``` r
createAbbrList(
  headings = list(Akronyme = c("Abkuerzung", "Bedeutung"), `Statistische Formelzeichen` =
    c("Symbol", "Bedeutung"))
)
```

## Arguments

- headings:

  Headings.

## Value

Returns a latex snippet.
