# Create variable information template.

Create variable information (background model, which variables in the
data set, ...) template.

## Usage

``` r
createVarInfo(
  GADSdat,
  inputForDescriptives,
  encodingList = NULL,
  makeStructure = FALSE
)
```

## Arguments

- GADSdat:

  `GADSdat` object.

- inputForDescriptives:

  Input for descriptive statistics calculation.

- encodingList:

  tbd.

- makeStructure:

  Should an automatic structuring of variables be created?

## Value

Returns the variable information template.
