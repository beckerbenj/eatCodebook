# Create structure template.

Create structure template based on variable information.

## Usage

``` r
createStructure(varInfo = NULL)
```

## Arguments

- varInfo:

  Object (either list or data.frame) containing variable information.

## Value

Structure template.

## Details

To create the structure template, information from the columns
`'Unterteilung.im.Skalenhandbuch'` and `'Gliederung'` are used.
